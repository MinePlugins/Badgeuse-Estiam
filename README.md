# Badgeuse-Estiam
Code de la Badgeuse ESTIAM

# Dépendences  :package:

- Python 3
  - Flask
  - Flask_SocketIO
  - graphqlclient
  - PySerial

# Matériels  :package:

- Ecran 7" HDMI
- Un Raspberry Pi
- Un Arduino
- Un RFID Reader RC522
- Des badges RFID (Mifare classic)
